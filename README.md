# Excel Sales Dashboard

## Project Overview

This project is an interactive Excel Sales Dashboard created using the Superstore Sales dataset.

## Features

* Sales by Region Analysis
* Profit by Category Analysis
* Monthly Sales Trend
* Interactive Slicers
* KPI Cards for Total Sales and Profit

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

## Insights

* West region generated highest sales.
* Technology category produced strong profits.
* Dashboard allows dynamic filtering using slicers.
